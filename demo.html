🛠️ Troubleshooting

Nếu gặp vấn đề:

  1. Không thể đăng nhập: Đảm bảo đã đăng ký tài khoản trước
  2. Dữ liệu bị mất: Clear browser data và đăng ký lại
  3. Giao diện lỗi: Refresh trang (F5)
  4. Mobile không đẹp: Kiểm tra viewport meta tag

📞 Hỗ trợ

Để có backend thật, bạn cần:

  • Deploy server lên Heroku, Vercel, hoặc Railway
  • Setup MongoDB database
  • Configure environment variables
  • Update API endpoints trong frontend
